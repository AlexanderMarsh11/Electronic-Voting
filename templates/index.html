Create Account

Register using your National ID. A PPK file will be generated for secure anonymous voting.

Host Election

Admin panel to start elections, set candidate filing, ballot opening, and ballot closing dates. Automatically tally and verify results.

File for Campaign

Declare your intention to run in a hosted election. Only residents of the election district can file unless it's a global election.

Vote

View active elections and all candidates. Cast your ballot securely using your PPK.

Results

View election results with cryptographic verification. Results are automatically tallied based on the election type (First Past the Post, Ranked Choice in future).
